Filter Syntax

Field Filters

hostname:api.example.com Exact or partial hostname match
host:cdn Shorthand for hostname
uri:/api/v2 URI path/query match
status:404 HTTP status code (exact or prefix e.g. 4)
method:POST HTTP method (GET, POST, PUT…)

Boolean Operators

api AND NOT cdn Must match "api", exclude "cdn"
google.com OR facebook.com Match either hostname
(status:4 OR status:5) AND uri:/auth Grouping with parentheses
NOT method:GET Exclude all GET requests

Free Text

analytics Matches hostname, URI, method, or status

Operators are case-insensitive

api and not cdn Same as AND / NOT
